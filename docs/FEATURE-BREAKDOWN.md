# 기능 분해 (상위 기능 / 하위 기능)

**기준 문서:** [PRD-block-html-dashboard.md](./PRD-block-html-dashboard.md)  
**작성일:** 2025-03-13

PRD의 기능 요구사항을 **상위 기능**과 **하위 기능**으로 계층 분해한 문서입니다.

---

## 1. 전체 레이아웃 (화면 구성)

| ID | 기능 | 구분 | 설명 |
|----|------|------|------|
| F1 | 전체 레이아웃 | 상위 | 3열 구조로 블록 팔레트·캔버스·결과 패널을 한 화면에 배치 |
| F1.1 | 3열 레이아웃 구조 | 하위 | 좌(팔레트) \| 중(캔버스) \| 우(결과 패널) 영역 구분 및 비율 유지 |
| F1.2 | 블록 팔레트 영역 | 하위 | 좌측 고정 너비, 세로 스크롤 가능한 패널 |
| F1.3 | 캔버스 영역 | 하위 | 중앙 드롭 존이 차지하는 영역 확보 |
| F1.4 | 결과 패널 영역 | 하위 | 우측 탭(Preview/HTML)이 차지하는 영역 확보 |
| F1.5 | 반응형 레이아웃 | 하위 | 1024px 이하에서 패널 비율/배치 조정 (Phase 3) |

---

## 2. 블록 팔레트

| ID | 기능 | 구분 | 설명 |
|----|------|------|------|
| F2 | 블록 팔레트 | 상위 | 좌측에 드래그 가능한 블록 목록을 카드 형태로 제공 |
| F2.1 | 블록 카드 UI | 하위 | 각 블록을 아이콘 + 블록 이름이 있는 카드/블록 형태로 표시 |
| F2.2 | 블록 드래그 가능 처리 | 하위 | 카드에 `draggable` 설정 및 드래그 시작 이벤트 처리 |
| F2.3 | 드래그 중 시각적 피드백 | 하위 | 드래그 중 반투명 또는 썸네일 등 시각적 피드백 |
| F2.4 | 블록 그룹(섹션) | 하위 | (선택) 기본/레이아웃/미디어 등 접을 수 있는 그룹으로 표시 |
| F2.5 | 이미지 블록 | 하위 | 이미지 블록 아이콘·라벨, 드롭 시 이미지용 테이블 HTML 생성 |
| F2.6 | 레이아웃 블록 | 하위 | 1열/2열/3열 컬럼 아이콘·라벨, 드롭 시 컬럼 수에 맞는 테이블 HTML |
| F2.7 | 텍스트 블록 | 하위 | 텍스트 블록 아이콘·라벨, 드롭 시 `<p>` 테이블 HTML |
| F2.8 | 버튼 블록 | 하위 | 버튼 아이콘·라벨, 드롭 시 `<a>` 테이블 HTML |
| F2.9 | 구분선 블록 | 하위 | 수평선 아이콘·라벨, 드롭 시 `<hr>` 테이블 HTML |
| F2.10 | 스페이서 블록 | 하위 | 간격 아이콘·라벨, 드롭 시 `height` 지정 빈 셀 테이블 HTML |

---

## 3. 캔버스 (드롭 존)

| ID | 기능 | 구분 | 설명 |
|----|------|------|------|
| F3 | 캔버스(드롭 존) | 상위 | 블록을 놓아 쌓는 중앙 작업 영역 |
| F3.1 | 드롭 존 영역 | 하위 | 블록을 드롭할 수 있는 영역 정의 및 드롭 이벤트 수신 |
| F3.2 | 드롭 가능 시 시각적 강조 | 하위 | 드래그 중 캔버스 위에 있을 때 테두리/배경 등 강조 |
| F3.3 | 블록 리스트 표시 | 하위 | 드롭된 블록을 위에서 아래 순서로 리스트 형태로 표시 |
| F3.4 | 빈 상태 플레이스홀더 | 하위 | 블록이 없을 때 "여기에 블록을 드래그하세요" 메시지 표시 |
| F3.5 | 블록 선택 UI | 하위 | (2차) 블록 선택 시 테두리·핸들·삭제 버튼 표시 |
| F3.6 | 캔버스 내 순서 변경 | 하위 | (2차) 캔버스에서 블록 드래그로 위/아래 순서 변경 |

---

## 4. 결과 패널 (Preview / HTML)

| ID | 기능 | 구분 | 설명 |
|----|------|------|------|
| F4 | 결과 패널 | 상위 | 우측에서 생성 결과를 Preview·HTML 탭으로 제공 |
| F4.1 | 탭 UI | 하위 | Preview / HTML 두 탭 전환 UI |
| F4.2 | Preview 탭 | 하위 | 캔버스 블록으로 생성한 HTML을 iframe 또는 sandbox div에 렌더링하여 미리보기 |
| F4.3 | HTML 탭 | 하위 | 생성된 전체 HTML 문자열(테이블 코딩) 표시 |
| F4.4 | HTML 포맷팅 | 하위 | HTML 탭에서 들여쓰기 등 가독성 포맷 적용 |
| F4.5 | 복사 버튼 | 하위 | HTML 탭에서 코드 복사 버튼 동작 |

---

## 5. 드래그 앤 드롭 동작

| ID | 기능 | 구분 | 설명 |
|----|------|------|------|
| F5 | 드래그 앤 드롭 | 상위 | 팔레트에서 캔버스로 블록을 드래그하여 추가하는 흐름 |
| F5.1 | 팔레트에서 드래그 시작 | 하위 | 블록 카드에서 드래그 시작 시 블록 타입 등 데이터를 dataTransfer에 설정 |
| F5.2 | 캔버스에 드롭 시 블록 추가 | 하위 | 캔버스 드롭 존에 드롭 시 해당 블록 타입 인스턴스를 캔버스 상태에 추가 |
| F5.3 | 드롭 후 자동 반영 | 하위 | 캔버스 상태 변경 시 Preview·HTML 탭에 즉시 반영 |
| F5.4 | 캔버스 내 드래그로 순서 변경 | 하위 | (2차) 캔버스 내 블록 드래그로 순서 변경 및 상태·화면 반영 |

---

## 6. 데이터 모델 및 HTML 생성

| ID | 기능 | 구분 | 설명 |
|----|------|------|------|
| F6 | 데이터 모델·HTML 생성 | 상위 | 캔버스 상태 관리 및 테이블 코딩 HTML 문자열 생성 |
| F6.1 | 캔버스 상태 관리 | 하위 | `CanvasBlock[]` 형태의 블록 배열 상태 추가·삭제·순서 변경 |
| F6.2 | 블록 타입·속성 정의 | 하위 | BlockBase 및 타입별 인터페이스(ImageBlock, TextBlock 등) 정의 |
| F6.3 | blocksToTableHtml() | 하위 | 블록 배열을 받아 테이블 코딩 전체 HTML 문자열을 반환하는 순수 함수 |
| F6.4 | 블록 타입별 테이블 스니펫 | 하위 | 이미지/레이아웃/텍스트/버튼/구분선/스페이서별 테이블 HTML 스니펫 매핑 |
| F6.5 | 최상위 table 래퍼 | 하위 | `role="presentation"`, `width="100%"`, `cellpadding="0"`, `cellspacing="0"` 등 적용 |
| F6.6 | 블록 속성 편집 반영 | 하위 | (2차) 이미지 URL, 텍스트, 링크 등 편집 시 상태 반영 후 HTML 재생성 |

## 7. 블록 속성 편집 (2차)

| ID | 기능 | 구분 | 설명 |
|----|------|------|------|
| F7 | 블록 속성 편집 | 상위 | 선택한 블록의 속성(URL, 텍스트, 링크 등)을 편집하는 기능 |

| F7.3 | 텍스트 블록 속성 편집 | 하위 | content 편집 |
| F7.4 | 버튼 블록 속성 편집 | 하위 | href, 라벨 등 편집 |
| F7.5 | 편집 결과 반영 | 하위 | 저장/적용 시 캔버스 상태 업데이트 및 Preview/HTML 갱신 |

## 12. 계층 구조 요약 (트리)

```
전체 레이아웃 (F1)
├── 3열 레이아웃 구조 (F1.1)
├── 블록 팔레트 영역 (F1.2)
├── 캔버스 영역 (F1.3)
├── 결과 패널 영역 (F1.4)
└── 반응형 레이아웃 (F1.5)

블록 팔레트 (F2)
├── 블록 카드 UI (F2.1)
├── 블록 드래그 가능 처리 (F2.2)
├── 드래그 중 시각적 피드백 (F2.3)
├── 블록 그룹 (F2.4)
└── 블록 종류: 이미지(F2.5), 레이아웃(F2.6), 텍스트(F2.7), 버튼(F2.8), 구분선(F2.9), 스페이서(F2.10)

캔버스 (F3)
├── 드롭 존 영역 (F3.1)
├── 드롭 가능 시 시각적 강조 (F3.2)
├── 블록 리스트 표시 (F3.3)
├── 빈 상태 플레이스홀더 (F3.4)
├── 블록 선택 UI (F3.5, 2차)
└── 캔버스 내 순서 변경 (F3.6, 2차)

결과 패널 (F4)
├── 탭 UI (F4.1)
├── Preview 탭 (F4.2)
├── HTML 탭 (F4.3)
├── HTML 포맷팅 (F4.4)
└── 복사 버튼 (F4.5)

드래그 앤 드롭 (F5)
├── 팔레트에서 드래그 시작 (F5.1)
├── 캔버스에 드롭 시 블록 추가 (F5.2)
├── 드롭 후 자동 반영 (F5.3)
└── 캔버스 내 순서 변경 (F5.4, 2차)

데이터 모델·HTML 생성 (F6)
├── 캔버스 상태 관리 (F6.1)
├── 블록 타입·속성 정의 (F6.2)
├── blocksToTableHtml() (F6.3)
├── 블록 타입별 테이블 스니펫 (F6.4)
├── 최상위 table 래퍼 (F6.5)
└── 블록 속성 편집 반영 (F6.6, 2차)

블록 속성 편집 (F7, 2차) · 블록 삭제 (F8, 2차) · 저장·불러오기 (F9, Phase3) · 반응형·접근성 (F10, Phase3) · 추가 블록 타입 (F11, Phase3)
```
